# 🏪 Sistema de Controle de Estoque

Sistema web moderno para gerenciamento de estoque com interface intuitiva e responsiva.

## ✨ Funcionalidades

- ✅ **Cadastro de Produtos** - Adicione produtos com nome e quantidade
- 🔍 **Busca em Tempo Real** - Filtre produtos instantaneamente
- ✏️ **Edição em Linha** - Modifique quantidades diretamente
- 🗑️ **Exclusão Segura** - Remove produtos com confirmação
- 📱 **Design Responsivo** - Funciona em desktop e mobile

## 🚀 Tecnologias Utilizadas

- **HTML5** - Estrutura semântica
- **CSS3** - Variáveis CSS, Flexbox, Grid, Responsividade
- **JavaScript ES6+** - Modules, Classes, Event Delegation
